# YouTube Trending Videos Data Analysis & Dashboard

This project analyzes YouTube Trending Videos data to uncover patterns behind video performance, audience engagement, and content popularity. The dataset was obtained from Kaggle and processed using Python, where Pandas was used for data cleaning, transformation, and exploratory data analysis (EDA). Missing values were handled, date fields were converted to proper datetime formats, and additional analytical features such as engagement rate and publishing hour were created to better understand content performance.

After cleaning and preparing the dataset, the data was stored and queried using PostgreSQL to perform business-oriented analysis. SQL queries were used to answer practical analytical questions such as identifying the most popular video categories, determining which channels generate the highest views, analyzing engagement patterns, and discovering the optimal publishing time for videos.

Finally, the processed data was connected to Power BI to build an interactive dashboard. The dashboard includes key performance indicators such as total views, total likes, number of trending videos, and average engagement rate. It also contains visualizations that highlight category performance, yearly trends in views, relationships between views and likes, top performing videos, and the impact of publishing time on video performance. Interactive slicers allow users to filter the dashboard by category and publishing hour for deeper exploration.

This project demonstrates an end-to-end data analytics workflow including data cleaning, exploratory data analysis, SQL-based business analysis, and interactive data visualization. The goal of the project is to extract meaningful insights from raw data and present them through a clear and interactive dashboard that can support data-driven decision making.
